# 🗄️ SQL Business Analytics Project

## Project Overview

This project demonstrates business data analysis using MySQL and the Global Superstore dataset. SQL queries were used to analyse sales performance, profitability, customer behaviour, product performance, and regional trends.

---

## Business Problem

Business users require meaningful insights from sales data to support better decision-making. This project answers common business questions using SQL by analysing sales, profit, customers, products, and regions.
